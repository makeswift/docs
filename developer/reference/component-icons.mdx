---
title: "Component icons"
description: Icons that can be used when registering a component in Makeswift.
---

When registering a component in Makeswift using [registerComponent](/developer/reference/runtime/register-component) you can choose an icon to go with it by including the `icon` prop.

```ts
runtime.registerComponent(Image, {
  type: "my-image",
  label: "My Image",
  props: {},
  icon: "image",
});
```
